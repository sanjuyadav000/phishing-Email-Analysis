# Phishing Email Analysis

## Objective
Identify phishing characteristics in a suspicious email sample.

## Tools Used
- Sample phishing email
- GitHub
- Text Editor

## Files Included
- phishing_email.txt
- report.md

## Phishing Indicators
- Spoofed sender address
- Urgent language
- Suspicious URL
- Generic greeting
- Social engineering attempt

## Conclusion
The email was identified as a phishing attempt based on multiple phishing indicators.
